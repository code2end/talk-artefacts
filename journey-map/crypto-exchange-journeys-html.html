üìö Domain Glossary

KYC (Know Your Customer)
Identity verification process requiring users to submit government-issued documents to enable full trading capabilities
OTC (Over-The-Counter)
Peer-to-peer trading where users buy/sell cryptocurrency directly with each other using fiat currency payments
MemberWallet
Digital wallet entity storing cryptocurrency balances with available and frozen amounts for each supported coin
Advertise
Trading offer created by members specifying price, limits, and payment methods for buying or selling cryptocurrency
Order Status
Transaction states: NONPAYMENT (awaiting payment), PAID (payment made), COMPLETED (crypto transferred), APPEAL (dispute)
Frozen Balance
Cryptocurrency amount locked during active orders or pending withdrawals, preventing double-spending
RealNameStatus
KYC verification states: NOT_CERTIFIED, PENDING (under review), VERIFIED (approved), FAILED (rejected)
MemberTransaction
Audit record of all financial activities including deposits (RECHARGE), withdrawals, trades, and fees
Wallet RPC
Remote Procedure Call service interfacing with blockchain networks for address generation and transaction broadcasting
Commission (jyRate)
Platform fee charged on OTC trades, calculated as percentage of transaction amount
QueryDSL
Type-safe SQL query builder used for complex database operations and dynamic filtering
Optimistic Locking
Concurrency control using version numbers to prevent conflicting updates to wallets and orders
Cold Wallet
Offline cryptocurrency storage for large amounts, requiring manual approval for withdrawals
Hot Wallet
Online wallet with automated withdrawal capabilities for smaller amounts below threshold
Aliyun OSS
Alibaba Cloud Object Storage Service used for storing KYC documents and user uploads
Kafka
Message queue system for asynchronous event processing and real-time notifications across services
